성균관대 정보 통신 학부 "프로그래밍 랩"에 등록한 학생들에게 제공되는 주요 프로그램입니다.

필요하지 않은 주석을 삭제하십시오.
필요한 코멘트를 추가하십시오.

이 프로그램(ckt.c)이 작동하도록 기능을 추가하십시오.
- 텍스트 파일을 입력받는다.
	정상 텍스트 파일 : "cktdata.txt"
	오류가 있는 텍스트 파일 : "ckterror" 폴더에 존재하는 텍스트 파일들
- 정상 텍스트 파일을 받아들였을 때와 오류가 있는 텍스트 파일을 받아들였을 때 정상 결과를 출력하거나 오류 메세지를 출력한다.
(자세한 내용은 "2017311583.정해진.HW5.pdf" 참고)

[주의]
프로그램 작동 시 입력할 텍스트 파일이 프로그램 파일이 존재하는 폴더에 포함되도록 하십시오.
프로그램 작동 시 헤더 파일에 "memcheck.h", "circuit.h"을 추가하십시오.
