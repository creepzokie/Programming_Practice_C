프로그래밍 실험실 배열 (및 포인터)

성균관대 정보 통신 학부 "프로그래밍 랩"에 등록한 학생들에게 제공되는 주요 프로그램입니다.

필요하지 않은 주석을 삭제하십시오.
필요한 코멘트를 추가하십시오.

이 프로그램(shortcut.c)이 작동하도록 기능을 추가하십시오.
- 파일 이름을 입력받는다.
- 링크 주소 저장 list의 버퍼 사이즈를 입력받는다.
- lnk가 파일을 가리키는지 폴더를 가리키는지 표시하고, 해당하는 주소를 출력한다.
(모범 출력 결과 : "정상_출력(fileLink).txt", "정상_출력(folderLink).txt" 참고)
- 입력한 파일 이름, 입력한 버퍼 사이즈에 이상이 있다면 오류 메세지 출력
(모범 출력 결과 : "bufferSize_작은경우.txt", "shortcutFile_아닌경우.txt" 참고)

[주의]
프로그램 작동 시 lnk 파일이 프로그램 파일이 존재하는 폴더에 포함되도록 하십시오.
