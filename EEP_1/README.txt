이 함수는 다음을 수행합니다.

(1) 0 ~ 90 도의 사인 함수 값을 계산하십시오. 
    각도는 "num_x_samples"로 나뉩니다. 
    예를 들어, num_x_samples가 31 인 경우 0 ~ 90도까지 31 개의 x 축 점이 사용됩니다. 
    즉 0과 90도를 포함하여 3도마다 사인 값이 계산됩니다.

(2) 계산 된 값은 2 차원 배열로 저장되며 "char graph [] [70]"은 1과 0으로 저장됩니다.
    해당 배열의 두 번째 인덱스는 사인 값에 사용되므로, 70은 사인 값의 최대 값 인 1.0을 의미합니다.
