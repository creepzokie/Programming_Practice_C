Compiler, Web Browser, PDF Reader, internet 연결 상태 등, 시험에 사용할 PC를 점검하세요. 시험 도중 power line을 건드리지 마세요. PC가 down되면, 그동안 작성한 program file을 잃게 됩니다.

quiz2.zip (이것 하나만 download 받으면 됩니다.)

  1. QUIZ #2에서 무었을 해야 하는지 알아보려면, 이 file을 참조하세요.
  2. q2.c와 memcheck.h를 사용하여 program 합니다.
    q2.c를 학번.이름.Q2.c로 시험 제출 게시판에 upload 합니다.
  3. screen.txt : 기대하는 출력
  • Console screen에 나타나는 출력은 위 screen.txt와 똑같아야 합니다. main()의 code를 건드리지 마세요.
  • File "q2.c"에 code를 추가하는 것만 허용할 뿐, 이미 존재하는 code를 제거하거나 수정할 수 없습니다.
  • Dynamic memory allocation에 "mymalloc()/myfree()"를 사용해야 합니 다. "malloc()/free()"를 사용할 수 없습니다.
  • Storage class "auto"인 memory를 필요 이상으로 사용하면 감점됩니다.
    에를 들어, array data는 dynamic memory allocation으로 사용해야 합니다.
  1. 만의 하나, 다운로드 받은 zip 파일을 풀지 않고 그냥 사용하고 있는 학생이 있다면, 지금 바로 zip 파일을 풀어서 사용하세요. 말썽의 원인이 될 수 있습니다.
  2. PC실에서의 시험이라고 해서, 다른 시험과 다를 것이 없습니다. 상식적인 수준에서 행동하시기 바랍니다. 부정행위에 대하여 엄격하게 처리합니다. 질서 유지와 부정행위에 대하여 조교 선생님이 전권을 행사합니다. 조교 선생님이 질서 유지에 협조하지 않은 학생이라고 보고하시면, 부정 행위를 사유로 F학점을 부여할 것입니다.
  3. 모든 통신 기기의 전원을 꺼서 가방 속에 넣으세요. 시험이 종료될 때까지 전원을 켜거나 눈에 띄면, 부정행위에 사용했건 아니건 상관없이 부정행위로 간주하여 처리합니다. USB 메모리 장치 또한 같습니다.
  4. 필요할 때, 화장실에 다녀올수 있으나, 조교 선생님에게 신고해야 합니다. 신고 없이 가면, 시험을 마친 것으로 간주하며, 자리로 돌아올 수 없습니다. 동시에 2명 이상이 갈 수 없습니다. 돌아오면 다른 사람이 갈 수 있도록 조교 선생님에게 신고하세요. 돌아와서 신고하지 않으면 시험을 0점 처리합니다.
